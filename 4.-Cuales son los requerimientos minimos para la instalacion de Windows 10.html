4.- Cuales son los requerimientos minimos para la instalacion de Windows 10?

  • Procesador: 1 GHz o más rápido
  • RAM: 1 GB para 32 bits o 2 GB para 64 bits
  • Espacio en disco duro: 16 GB para SO de 32 bits o 32 GB para SO de 64 bits
  • Tarjeta gráfica: DirectX 9 o posterior con controlador WDDM 1.0
  • Pantalla: 800x600
  • Conexión a Internet: Se requiere conexión a Internet para realizar actualizaciones y aprovechar algunas características.
