33. De que manera puedo asignar una dirección Ip en Windows

Cómo asignar una dirección IP en Windows

Asignar una dirección IP a través del Panel de control

Para asignar una dirección IP manualmente en Windows utilizando el Panel de control, sigue estos pasos:

Paso 1: Abrir el Panel de control

  • Presiona las teclas Win + R, escribe control y presiona Enter para abrir el Panel de control.

Paso 2: Acceder a las Conexiones de red

  • En el Panel de control, selecciona Redes e Internet.
  • Haz clic en Centro de redes y recursos compartidos.
  • En el panel izquierdo, haz clic en Cambiar configuración del adaptador.

Paso 3: Configurar las propiedades del adaptador

  • Haz clic derecho en la conexión de red que deseas configurar y selecciona Propiedades.
  • Selecciona Protocolo de Internet versión 4 (TCP/IPv4) y haz clic en Propiedades.

Paso 4: Asignar una dirección IP

  • En la ventana de Propiedades de TCP/IPv4, selecciona Usar la siguiente dirección IP.
  • Ingresa la dirección IP deseada, la máscara de subred y la puerta de enlace predeterminada.
  • Opcionalmente, puedes ingresar las direcciones de servidor DNS preferido y alternativo.
  • Haz clic en Aceptar para aplicar los cambios.

Asignar una dirección IP a través del Símbolo del sistema

También puedes asignar una dirección IP manualmente utilizando el Símbolo del sistema. Sigue estos pasos:

Paso 1: Abrir el Símbolo del sistema como administrador

  • Presiona las teclas Win + X y selecciona Símbolo del sistema (Administrador) o Windows PowerShell (Administrador).

Paso 2: Usar el comando netsh

  • Escribe el siguiente comando para asignar una dirección IP: 
    netsh interface ip set address name="NombreConexión" static DirecciónIP MáscaraSubred PuertaEnlace

    Por ejemplo:

    netsh interface ip set address name="Ethernet" static 192.168.1.100 255.255.255.0 192.168.1.1
  • Para asignar servidores DNS, usa el siguiente comando: 
    netsh interface ip set dns name="NombreConexión" static DirecciónDNS

    Por ejemplo:

    netsh interface ip set dns name="Ethernet" static 8.8.8.8

Conclusión

Asignar una dirección IP manualmente en Windows puede ser útil en varias situaciones, como configurar redes domésticas o solucionar problemas de conectividad. Puedes hacerlo a través del Panel de control o utilizando el Símbolo del sistema, dependiendo de tus preferencias y necesidades.
