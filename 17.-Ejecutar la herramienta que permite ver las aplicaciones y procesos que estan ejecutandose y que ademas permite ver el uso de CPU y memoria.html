17.- Ejecutar la herramienta que permite ver las aplicaciones y procesos que estan ejecutandose y que ademas permite ver el uso de CPU y memoria

Ver Aplicaciones y Procesos en Ejecución con el Administrador de tareas

Para ver las aplicaciones y procesos que están ejecutándose en tu computadora, así como el uso de CPU y memoria, puedes utilizar el Administrador de tareas en Windows 10. A continuación se muestran los pasos para ejecutar esta herramienta:

Pasos para Ejecutar el Administrador de tareas

Abrir el Administrador de tareas

  1. Presiona Ctrl + Shift + Esc para abrir el Administrador de tareas directamente.
  2. También puedes presionar Ctrl + Alt + Delete y luego seleccionar Administrador de tareas.
  3. Otra opción es hacer clic derecho en la barra de tareas y seleccionar Administrador de tareas.

Ver las aplicaciones y procesos

  1. Si el Administrador de tareas se abre en modo simplificado, haz clic en Más detalles en la parte inferior.
  2. En la pestaña Procesos, verás una lista de aplicaciones y procesos que están actualmente en ejecución.

Ver el uso de CPU y memoria

  1. En la pestaña Procesos, puedes ver columnas que muestran el uso de CPU, memoria, disco y red para cada proceso.
  2. También puedes ir a la pestaña Rendimiento para ver gráficos en tiempo real del uso de CPU, memoria, disco, red y GPU.
