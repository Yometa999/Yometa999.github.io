24.- En el Simbolo del sistema cual es el comando que me permite asignar o quitar atributos de un archivo ejemplificar ocultando un archivo

Comando attrib en el Símbolo del sistema

El comando attrib permite asignar o quitar atributos de un archivo o directorio en el sistema operativo Windows. A continuación, se explica cómo utilizar este comando y se proporciona un ejemplo para ocultar un archivo.

Paso 1: Abrir el Símbolo del sistema

Puedes hacerlo buscando "cmd" en el menú de inicio y seleccionando "Símbolo del sistema".

Paso 2: Usar el comando attrib

La sintaxis básica del comando attrib es la siguiente:

attrib [+atributo | -atributo] [ruta_del_archivo]

Donde +atributo agrega un atributo y -atributo quita un atributo. Los atributos comunes son:

  • H - Oculto
  • R - Solo lectura (Read-only)
  • A - Archivo (Archive)
  • S - Sistema

Ejemplo: Ocultar un archivo

Para ocultar un archivo llamado ejemplo.txt en el directorio C:\Documentos, usa el siguiente comando:

attrib +H C:\Documentos\ejemplo.txt

Este comando asigna el atributo "Oculto" al archivo ejemplo.txt.

Ejemplo: Mostrar un archivo oculto

Para quitar el atributo "Oculto" del mismo archivo, usa el siguiente comando:

attrib -H C:\Documentos\ejemplo.txt

Este comando quita el atributo "Oculto" del archivo ejemplo.txt, haciéndolo visible nuevamente.

Conclusión

El comando attrib es útil para cambiar los atributos de archivos y directorios en Windows. Puedes usarlo para ocultar archivos, hacerlos de solo lectura, y más.
